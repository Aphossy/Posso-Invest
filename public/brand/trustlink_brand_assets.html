06 - Brand Usage Guidelines
Logo Use
DO Use the navy version on white/cream backgrounds
DO Use the reversed version on dark backgrounds
DON'T Stretch, recolor, or separate the icon from wordmark
DON'T Place on busy or patterned backgrounds
Color Rules
DO Pair Navy Deep with Gold Core as primary combination
DO Use Cream Warm as the default page/document background
DON'T Use Gold Core on white without navy context
DON'T Use more than 3 colors in any single document
Typography
DO Use Cormorant Garamond for all headings and titles
DO Use DM Mono for IDs, references, and data fields
DON'T Mix more than two font families in one document
DON'T Use Cormorant Garamond at sizes below 14px
